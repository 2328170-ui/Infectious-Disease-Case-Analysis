# Infectious Disease Case Analysis

A comprehensive exploratory data analysis (EDA) and predictive modeling project on early-stage infectious disease data, developed on behalf of **HealthGuard Analytics Pvt. Ltd.**

The analysis aims to provide actionable insights to national public health authorities to optimize screening processes, isolation planning, and early response systems.

---

## Objective

- Analyze patterns in early-stage infectious disease case data
- Build predictive models to support early detection and outbreak management
- Deliver data-driven recommendations to improve public health response

---

## Features

### Exploratory Data Analysis
- Distribution analysis of key epidemiological features
- Correlation heatmaps and statistical summaries
- Identification of high-risk patient segments

### Data Preprocessing
- Handling missing values and outliers
- Feature encoding and normalization
- Train-test splitting for model evaluation

### Predictive Modeling
- Classification models to predict disease outcomes
- Model performance comparison (accuracy, precision, recall, F1-score)
- Feature importance analysis

### Insights & Recommendations
- Patterns supporting screening optimization
- Risk stratification for isolation planning
- Early warning indicators for health authorities
